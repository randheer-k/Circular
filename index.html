  • Only the best of Buffer's features

    Circular implements Buffer's great original MVP: the ability to stock up tweets, choose a schedule, and have them shared automatically through the day.

  • Built on Backbone, Bootstrap and MongoDB

    Circular is a Backbone.js application that communicates with a MongoDB datastore through a REST API written in PHP.

  • And it's on Github!

    Circular is developed in the open on Github by Julien Chaumond. Feel free to fork and contribute!
